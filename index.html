Cascading Style Sheet atau CSS adalah bahasa pemograman berbasis aturan yang dikembangkan oleh World Wide Web Consortiom (W3C) pada tahun 1996. CSS dikembangkan karena HTML yang berfungsi untuk memformat halaman web tidak bisa menambahkan gaya atau tata letak tingkat lanjut. Misalnya, pada HTML versi 3.2, sudah diperkenalkan tag < font > agar gaya teks bisa diubah. Namun, web designer amatir menyalahgunakannya dan tidak ada opsi fallback seperti yang sekarang tesedia di CSS. Jadi, pilihan terbaiknya adalah tetap memisahkan pemformatan (THML) dan penataan gaya (CSS).
